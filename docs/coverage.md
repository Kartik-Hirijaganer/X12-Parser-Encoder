# Coverage

Coverage is enforced in CI with staged Phase 8 thresholds:

- Library: `>= 90%`
- API: `>= 85%`
- Web: generated by `vitest --coverage` and reported for drift detection

Last verified locally on `2026-04-12`:

- Library: `90.04%`
- API: `90.59%`
- Web statements: `62.16%`

Run the full coverage gate locally with:

```bash
make coverage
```

The badge at `docs/coverage-badge.svg` is intended to be refreshed by the Phase 8 coverage workflow or the `/check-coverage` command after coverage reports are regenerated.
